Detailed Description

Modbus RTU server implementation for temperature monitoring.

Provides Modbus RTU communication interface for reading temperature values, status information, and configuring alarm thresholds. Supports function codes 03 (Read Holding Registers), 06 (Write Single Register), and 16 (Write Multiple Registers).
