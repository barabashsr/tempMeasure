Detailed Description

Main alarm class for temperature monitoring system.

Manages individual alarm instances with full lifecycle tracking, priority handling, and integration with measurement points

Constructor for creating a new alarm.

Constructor - Creates a new alarm instance.

Parameters
[in]typeThe type of alarm being created
[in]sourcePointer to the measurement point that triggered this alarm
[in]priorityPriority level for this alarm (default: PRIORITY_MEDIUM)
Note
The alarm is created in NEW stage and timestamp is set to current time
Parameters
[in]typeType of alarm being created
[in]sourcePointer to measurement point that triggered the alarm
[in]priorityPriority level for the alarm

Initializes alarm with NEW stage, current timestamp, and default timeouts. Generates configuration key and logs alarm creation event.
